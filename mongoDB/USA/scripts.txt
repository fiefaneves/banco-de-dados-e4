Passo 1: Preparação do Ambiente (Pré-requisitos)

Temos duas maneiras:

1. INSTALANDO:
    - Antes de começar, você precisa ter o MongoDB instalado e funcionando na sua máquina.

    - Instale o MongoDB Community Server: Se ainda não o fez, baixe e instale o servidor do MongoDB. Ele é o próprio banco de dados que rodará em segundo plano.

    - Instale o MongoDB Shell (mongosh): Esta é a ferramenta de linha de comando que você usará para se conectar ao seu banco de dados e executar os scripts.

    - Verifique a Instalação: Abra o seu terminal ou prompt de comando e digite mongosh. Se a instalação foi bem-sucedida, você se conectará ao seu servidor local do MongoDB. Digite .exit para sair.

2. MONGODB ATLAS (ONLINE):
    - Vamos criar e configurar um banco de dados gratuito na nuvem.

    - Crie uma Conta Gratuita no MongoDB Atlas:
        - Acesse o site: MongoDB Atlas

        - Faça seu cadastro. Usar uma conta do Google ou similar acelera o processo.

    - Crie um Cluster Gratuito (Banco de Dados):
        - Após o login, a plataforma irá te guiar para criar um novo cluster.

        - Escolha a opção "Shared" (Compartilhado), que inclui o plano gratuito M0 Sandbox.

        - Provedor de Nuvem e Região: Pode manter os padrões (AWS). Para melhor performance, na Região, procure por uma próxima a você, como sa-east-1 (São Paulo).

        - Clique em "Create Cluster". A criação do seu banco de dados levará de 3 a 5 minutos.

    - Configure a Segurança de Acesso:
        - Usuário do Banco de Dados: No menu à esquerda, vá em Database Access -> + ADD NEW DATABASE USER. Crie um nome de usuário e uma senha. Anote-os bem.
            - Exemplo: Usuário: aluno, Senha: projeto123

        - Acesso por IP: No menu à esquerda, vá em Network Access -> + ADD IP ADDRESS.
            - Clique em ADD CURRENT IP ADDRESS para permitir o acesso da sua rede atual.

            - Se preferir, pode clicar em ALLOW ACCESS FROM ANYWHERE (0.0.0.0/0) para acessar de qualquer lugar (menos seguro, mas prático para projetos acadêmicos).

            - Clique em Confirm.

    - Instale Apenas o MongoDB Shell (mongosh):
        - Você ainda precisa da ferramenta de linha de comando para executar os scripts. Ela é leve e a instalação é rápida.

Passo 2: Execução dos Scripts

- Abra seu terminal ou prompt de comando.

- Navegue até a pasta raiz do seu projeto (mongoDB).

- Execute cada script usando o mongosh com o seguinte comando:

# Para executar o cenário 1
mongosh --file scripts/cenario1_referencia_1_1.js

# Para executar o cenário 2
mongosh --file scripts/cenario2_embutido_1_1.js

# Para executar o cenário 3
mongosh --file scripts/cenario3_referencia_N_N.js

# Para executar o cenário 4
mongosh --file scripts/cenario4_embutido_N_N.js

Ao executar cada comando, você verá no terminal a saída que incluímos com os comandos print(), mostrando o resultado de cada consulta.

Verificação Manual (Opcional)
Para ter certeza de que os dados foram criados corretamente, você pode se conectar ao banco de dados e verificar as coleções manualmente.

Passe 3 (Opcional): 

- No terminal, digite mongosh.

- Conecte-se ao banco de dados do projeto: use fabrica_de_chocolate

- Liste todas as coleções: show collections

- Consulte uma coleção para ver os dados: db.chocolates_ref.find().pretty()
